PHP (Hypertext Preprocessor)

Pengertian PHP

PHP adalah singkatan dari "Hypertext Preprocessor", yaitu bahasa pemrograman yang digunakan terutama untuk pengembangan web. PHP adalah bahasa server-side yang artinya kode PHP dijalankan di server dan hasilnya dikirimkan ke browser sebagai HTML.

Sejarah PHP

PHP pertama kali dibuat oleh Rasmus Lerdorf pada tahun 1994 dan awalnya bernama "Personal Home Page". Seiring waktu, PHP berkembang menjadi bahasa pemrograman yang populer untuk pengembangan web dinamis dan sekarang digunakan oleh jutaan situs web di seluruh dunia.

Fungsi dan Kegunaan PHP

PHP memiliki banyak fungsi dan kegunaan dalam pengembangan web, antara lain:

  • Menghasilkan konten dinamis di halaman web.
  • Mengelola data dari formulir HTML.
  • Berinteraksi dengan database.
  • Mengontrol sesi pengguna.
  • Mengakses dan memanipulasi file di server.

Syntax Dasar PHP

Berikut adalah contoh sintaks dasar dari PHP:

                
<?php
    echo "Hello, World!";
?>

Contoh di atas akan menampilkan teks "Hello, World!" di halaman web.

Variabel PHP

Variabel dalam PHP digunakan untuk menyimpan data yang dapat diubah selama skrip dijalankan. Variabel dalam PHP diawali dengan tanda dolar ($) diikuti dengan nama variabel. Berikut adalah contoh deklarasi variabel dalam PHP:

                
<?php
    $nama = "Arifin";
    $umur = 20;
    $tinggi = 170.5;
    echo "Halo Nama Saya {$nama}, Umur Saya {$umur} tahun, dan Tinggi Saya {$tinggi} cm.";
?>

Contoh di atas mendeklarasikan tiga variabel: $nama, $umur, dan $tinggi.

Operator PHP

Operator dalam PHP digunakan untuk melakukan operasi pada variabel dan nilai. Berikut adalah beberapa operator dasar dalam PHP:

  • Operator Aritmatika: +, -, *, /, %
  • Operator Penugasan: =, +=, -=, *=, /=, %=
  • Operator Perbandingan: ==, !=, >, <, >=, <=
  • Operator Logika: &&, ||, !

Berikut adalah contoh penggunaan operator aritmatika dalam PHP:

                
<?php
    $a = 10;
    $b = 20;
    $c = $a + $b;
    echo $c; // Output: 30
?>

Pengkondisian PHP

Pengkondisian dalam PHP digunakan untuk membuat keputusan berdasarkan kondisi tertentu. Struktur pengkondisian yang paling umum digunakan adalah pernyataan if, else if, dan else. Berikut adalah contoh pengkondisian dalam PHP:

                
<?php
    $nilai = 85;
    if ($nilai >= 90) {
        echo "A";
    } elseif ($nilai >= 80) {
        echo "B";
    } elseif ($nilai >= 70) {
        echo "C";
    } elseif ($nilai >= 60) {
        echo "D";
    } else {
        echo "E";
    }
?>

Contoh di atas akan menampilkan "B" karena nilai adalah 85.

Looping PHP

Looping digunakan untuk menjalankan blok kode berulang kali. PHP mendukung beberapa jenis looping, seperti for, while, do-while, dan foreach. Berikut adalah contoh penggunaan loop for dalam PHP:

                
<?php
    for ($i = 0; $i < 5; $i++) {
        echo "Nomor: $i <br>";
    }
?>

Contoh di atas akan menampilkan angka 0 sampai 4 di halaman web.

Array PHP

Array adalah tipe data yang dapat menyimpan beberapa nilai dalam satu variabel. Berikut adalah contoh deklarasi dan penggunaan array dalam PHP:

                
<?php
    $buah = array("Apel", "Mangga", "Jeruk");
    echo "Saya suka " . $buah[1];
    // Output: Saya suka Mangga
?>

Contoh di atas mendeklarasikan array bernama $buah yang berisi tiga elemen: "Apel", "Mangga", dan "Jeruk".

Fungsi PHP

Fungsi dalam PHP digunakan untuk mengelompokkan kode yang dapat digunakan kembali. Fungsi dapat menerima parameter dan mengembalikan nilai. Berikut adalah contoh deklarasi dan penggunaan fungsi dalam PHP:

                
<?php
    function salam($nama) {
        return "Halo, $nama!";
    }
    echo salam("Adit");
    // Output: Halo, Adit!
?>

Contoh di atas mendeklarasikan fungsi bernama salam yang menerima satu parameter $nama dan mengembalikan teks sapaan.

$_SESSION

$_SESSION digunakan untuk menyimpan data sesi yang dapat diakses di seluruh halaman dalam aplikasi:

                
<?php
session_start();
$_SESSION["username"] = "john_doe";
$_SESSION["email"] = "john@example.com";
echo "Username: " . $_SESSION["username"] . "<br>";
echo "Email: " . $_SESSION["email"];
?>

$_COOKIE

$_COOKIE digunakan untuk menyimpan data kecil di sisi klien yang dapat diakses di beberapa halaman:

                
<?php
setcookie("user", "John Doe", time() + (86400 * 30), "/");
if(isset($_COOKIE["user"])) {
    echo "User: " . $_COOKIE["user"];
} else {
    echo "Cookie 'user' tidak ditemukan!";
}
?>

Ini merupakan contoh project sederhana dari beberapa fungsi seperti get, post, session dan cookie.

File Handling PHP

File handling dalam PHP digunakan untuk membaca, menulis, dan mengelola file di server. Berikut adalah contoh cara membuka dan membaca file dalam PHP:

                
<?php
    $file = fopen("contoh.txt", "r") or die("Tidak bisa membuka file!");
    echo fread($file, filesize("contoh.txt"));
    fclose($file);
?>

Contoh di atas membuka file bernama contoh.txt dalam mode baca, membaca isinya, dan menutup file tersebut.
